# Roadmap

Ashbound is built in public. This is a rough direction, not a promise of dates.

## Shipped

- Core gameplay loop — movement, auto-attacking weapons, XP, level-ups
- Mobile controls (virtual joystick), HUD, portrait mode
- 6-weapon system with multi-slot loadouts
- 5 locations with distinct enemies, bosses, and difficulty curves
- 13 enemy types + multi-phase boss fights
- Homing projectiles and per-weapon behaviors
- Onchain NFT badges (ERC-721) on Base mainnet
- Daily onchain check-in with streak tracking
- Builder Code attribution on all transactions
- Wallet integration (Coinbase, MetaMask, Rabby, injected)

## In Progress

- Background music and sound effects
- Badge rarity tiers
- In-game badge inventory

## Exploring

- Token drops and rewards
- Additional locations and bosses
- Pixel-font logo and UI polish
- Leaderboards
