## ИДЕИ ОТ ПОЛЬЗОВАТЕЛЯ (пиксельный RPG-стиль интерфейса)

Общая цель: весь интерфейс — кнопки, надписи, бейджи — в едином
пиксельном RPG-стиле. Графику пользователь генерирует сам (нейронки),
Claude даёт промпты и подключает в код.

### 🔜 Пиксельные RPG-кнопки
- Пользователь генерирует НЕ кнопку целиком, а рамку для 9-slice
  (маленький PNG рамки; CSS border-image растягивает под любой текст).
- Нужны 2 состояния: обычная + нажатая.
- Claude: даёт промпт, переписывает globals.css + компоненты под 9-slice.
- ⚠️ Зависит от ассетов пользователя — код пишется ПОСЛЕ получения картинок.

### 🔜 Логотип ASHBOUND пиксельным шрифтом (главное меню)
- Вариант A: готовый бесплатный пиксельный шрифт — надёжно.
- Вариант B: логотип как сгенерированная картинка.
- ⚠️ ИИ часто перевирает буквы в тексте — вариант B может потребовать
  несколько перегенераций или ручной правки букв.
- Claude: даёт промпт / подключает шрифт.

### 💡 Бейджи за run разной рарности
- Статичные картинки бейджей — ИИ-генерация подходит (не анимация).
- ⚠️ Рарность должна определяться В СМАРТ-КОНТРАКТЕ (например по score),
  иначе её можно подделать. Значит — доработка контракта + редеплой.
- Claude: даёт промпты на бейджи; отдельно — правка контракта под рарность.

### 💡 Инвентарь бейджей в главном меню
- Показывать бейджи, которые игрок наминтил.
- ⚠️ Архитектурная развилка (решить до реализации):
  - читать реальные NFT с блокчейна (честное владение, сложнее);
  - либо хранить локально (просто, но не настоящее владение).
- Большая отдельная задача — детализировать перед стартом.

### 💡 Дроп «мелочи» юзерам за бейджи
- Токеномика / награды за владение бейджами.
- Серьёзная отдельная веха. Пока только зафиксировано как намерение,
  детали не проработаны.
